# Year 3 Information Visualisation Coursework

This repository contains coursework for **COMP3736 Information Visualisation**.

The project is a web-based information visualisation coursework project built with HTML. It focuses on presenting data through an interactive visual interface and supporting a user study-style evaluation process.

## Project Overview

The coursework includes a small web interface for displaying an information visualisation. The project structure includes separate HTML files for the page layout, shared header, and graph display.

The repository also contains coursework documentation and participant information materials used as part of the assessment.

## Repository Contents

| File | Description |
|---|---|
| `index.html` | Main entry page for the visualisation coursework |
| `header.html` | Shared header or page component |
| `show_graph.html` | Page used to display the graph or visualisation |
| `Example participant information sheet.docx` | Participant information document for the coursework evaluation |
| `IV cw 2 2023 v1b.docx` | Coursework specification document |

## How to View the Project

Clone the repository:

```bash
git clone https://github.com/ml20g2l/Year3-Information-Visualisation.git
```

Open the project folder:

```bash
cd Year3-Information-Visualisation
```

Open `index.html` in a web browser.

You can also open `show_graph.html` directly to view the graph display page.
